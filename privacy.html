# Privacy Policy Last updated: 2026 This application is used to publish media content to Instagram accounts via the official Instagram Graph API. ## Information We Collect The application does not collect, store, or process personal data on external servers. The app may access the following information through Facebook and Instagram authorization: * Instagram account ID * Media publishing permissions * Access tokens required for API authentication No passwords, private messages, contacts, or personal content are stored. ## How We Use Information The information obtained through the Instagram Graph API is used only for: * Publishing media content * Managing Instagram account access * Authenticating API requests ## Data Storage No personal data is stored permanently. Access tokens are used only for authentication purposes and can be revoked at any time by removing the application from your Facebook account settings. ## Data Deletion Users can request data deletion by: 1. Removing the application from Facebook Business Integrations 2. Sending a request to: [email@example.com](mailto:email@example.com) All access tokens will become invalid immediately after removal. ## Contact If you have any questions about this Privacy Policy, contact: [email@example.com](mailto:email@example.com)
